** 함수 명세

1. 생성자 : 해당 class의 생성자 함수
2. 양식 함수 : 입력을 받아 처리를 하는 함수
3. 출력 함수 : 출력만을 하는 함수
4. 반환 함수 : 데이터를 반환하는 함수


** Class Diagram

[class] Kiosk
- __init__()			필요한 객체 선언 및 멤버 변수 선언 		생성자
